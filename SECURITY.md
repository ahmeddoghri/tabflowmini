# Security

tabflowmini is a local research demo. It makes no network calls at runtime and does
not require API keys. The benchmark data is generated inside the repository.

If you find a security issue, please email contact@ideaboundary.com with a
short reproduction and the impact. Please do not open a public issue for a
vulnerability before I have had a chance to fix it.

Dependency updates are intentionally conservative. Runtime dependencies should
stay at zero unless a new dependency is clearly worth the maintenance cost.
